# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | yes       |

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub's **"Report a vulnerability"** button (Security tab) or email deependrsharma55@gmail.com.

Do not open a public issue for security reports.

Reports are acknowledged within 48 hours, with a resolution timeline provided after triage. Coordinated disclosure is followed.
